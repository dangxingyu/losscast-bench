# losscast-dev

Development scripts for data pipeline, conversion, and benchmark construction.
Not published to the main repo.

## Structure

```
convert/                    # Raw data → benchmark format conversion
  ncpl_to_bench.py              # NCPL HuggingFace dataset → data/staging/
  marin_raw_to_bench.py         # raw_data/marin/ WandB exports → benchmark format

explore/                    # Data exploration and sanity checks
  inspect_ncpl.py               # Load & explore NCPL dataset from HuggingFace
  inspect_marin_raw.py          # Explore raw_data/marin/ structure

splits/                     # Split construction
  build_splits.py               # Construct train/val splits (NCPL-style logic)

examples/                   # Real conversion test cases
  nanochat_d12/                 # nanochat 12-layer model
    source/                     # Original nanochat source (gpt.py, optim.py, base_train.py)
    expected/                   # Expected conversion output (config.json)
    converted/                  # Actual conversion output (generated by agent)
  nanochat_d24_muon/            # nanochat 24-layer speedrun model

agents/                     # Agent test harnesses
  convert_and_validate.md       # Prompt for Claude Code / Agent SDK conversion test
  run_conversion_test.sh        # Shell script to run conversion + validation end-to-end

PLANS.md                    # Development roadmap (test split, find_similar_runs API)
```

## Workflows

### Data pipeline (done)
1. `explore/inspect_ncpl.py` — understand NCPL dataset
2. `convert/ncpl_to_bench.py` — bulk convert to staging
3. `splits/build_splits.py` — assign to train/val
4. Validate with `scripts/validate_contribution.py`

### Testing the code-rewriter skill
```bash
# Automated: run Claude Code on nanochat source
bash losscast-dev/agents/run_conversion_test.sh

# Manual: give Claude Code the conversion prompt
claude "$(cat losscast-dev/agents/convert_and_validate.md)"
```
